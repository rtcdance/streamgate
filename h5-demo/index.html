Step 1

🌐 Backend Target

First make sure the demo points to the correct backend. Every later acceptance step depends on this.

Expected

The `Backend` status becomes `Online` and Step 1 turns green.

If the page is opened from a static server, the backend URL is usually `http://localhost:29090`.

🎨 NFT Verification

Expected

🎬 Protected Playback
